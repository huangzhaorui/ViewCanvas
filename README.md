# ViewCanvas

#### 介绍
一个canvas2d项目，提供了较为完善的事件监听、方便自定义的元素渲染和元素扩展，具有较好的性能。


#### 安装教程

##### 1.在vue项目中安装  
在项目根目录安装
```
npm i view-canvas
```
在vue的main.js中引入
```
import "view-canvas"
```
##### 2. 在html文件中引入
```
<script src="https://huang_zhao_rui.gitee.io/viewcanvas/ViewCanvas.js"/>
```
